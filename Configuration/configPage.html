User Ratings Plugin

Allow users to rate movies, TV shows, and music. Ratings are visible to all users on the server.
Notify users when someone rates an item
Display average user rating on item pages

API Endpoints

Rate an item: POST /UserRatings/Rate?itemId=xxx&rating=5¬e=Great!
Get item ratings: GET /UserRatings/Item/{itemId}
Get my rating: GET /UserRatings/MyRating/{itemId}
Get user ratings: GET /UserRatings/User/{userId}
Delete rating: DELETE /UserRatings/Rating?itemId=xxx
